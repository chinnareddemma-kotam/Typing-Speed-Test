# Typing-Speed-Test

Typing Speed Test is a web-based application designed to measure and improve typing performance through real-time speed and accuracy analysis. The application provides multiple difficulty levels with a collection of sentences and paragraphs, allowing users to practice typing in different scenarios.



The system calculates Words Per Minute (WPM), tracks typing accuracy, records mistakes, and stores the user's best score and typing history using browser local storage. With instant visual feedback and performance statistics, users can monitor their progress and enhance their typing skills effectively.



Built using HTML, CSS, and JavaScript, this project demonstrates front-end development concepts such as DOM manipulation, event handling, local storage management, and real-time data processing.



# Features

Multiple difficulty levels (Easy, Medium, Hard)

Custom timer options (15s, 30s, 60s, 120s)

Real-time character highlighting

Live WPM calculation

Accuracy tracking

Error counting

Best score storage using Local Storage

Typing history and leaderboard

Responsive user interface

Instant result popup
